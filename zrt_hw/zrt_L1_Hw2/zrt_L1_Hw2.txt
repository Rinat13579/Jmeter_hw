ДЗ-2: Научиться использовать отладчик JMeter
1. Установите менеджер плагинов в JMeter.
2. При помощи менеджера плагинов установите отладчик.
3. Выполните в отладчике ранее разработанный сценарий.
В качестве ответа на задание приложите архив со скриншотами, которые демонстрируют выполнение сценария в отладчике.


Для тестирования использовался инструмент JMeter, при помощи которого эмулировалась отправка запросов пользователями на сервер по протоколу HTTP. В качестве стенда использовалось прилежение Addressbook версии 4.1.4. установленное на виртуальной машине.
Тестирование выполнялось по одному сценарию: создание контакта в адрессной книге, с последующим возвращением на главную страницу. Данные для создания контакта генерируются автоматически. Отладка сценария проходила через отладчик Step-by-step Debugger, установленный через менеджер плагинов. Приложены скриншоты.
